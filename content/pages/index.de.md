Dies ist eine plain-Website — ein Ausgangspunkt, den Sie ganz zu Ihrem eigenen machen können. Sie ist bereits live, bereits schnell, und jedes Wort davon liegt in Markdown-Dateien, die Sie kontrollieren.

> Klartext rein, eine schnelle Website raus. Keine Datenbank, kein Dashboard zum Erlernen, nichts, was still ausfallen kann.

## Sehen Sie sich um

- Der **[Blog](/blog/)** zeigt, wie Beiträge aussehen — Text, Bilder und Formatierung.
- Die **[Über-Seite](/de/about/)** ist ein Platzhalter, der auf Ihre Geschichte wartet.
- Die **[Suche](/search/)** durchsucht die ganze Website, ganz ohne Server.

## Machen Sie sie zu Ihrer

Jede Seite ist eine Markdown-Datei in `content/`. Benennen Sie die Website in `site.config.json` um, schreiben Sie eine Seite neu, fügen Sie einen Beitrag hinzu — jedes Speichern ist ein Commit, und jede Version lässt sich wiederherstellen. Bearbeiten Sie im Browser unter `/admin/` oder in Ihrem eigenen Editor und pushen Sie; beides schreibt an denselben Ort.

Wenn Sie das gesamte Erscheinungsbild ändern möchten, besteht das Theme aus einer Handvoll CSS-Variablen. Löschen Sie diese Beispiele, wann immer Sie möchten — sie sind jeweils mit `example: true` markiert — und dies ist keine Demo mehr, sondern wird zu Ihrer Website.
